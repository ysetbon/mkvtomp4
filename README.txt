# MKV to MP4 Converter

A simple Windows application to convert MKV video files to MP4 format.

## Versions

- **Version 1.0** - Original release (231.7 MB)
- **Version 1.0.1** - Optimized smaller release (62.5 MB) with identical features

## Features

- Convert MKV files to MP4 format
- Batch conversion for multiple files
- Progress tracking
- Simple user interface

## Installation

1. Download the installer for your preferred version:
   - `MKVtoMP4Converter_Setup.exe` (v1.0)
   - `MKVtoMP4Converter_Setup_ver_1_01.exe` (v1.0.1)
2. Run the installer and follow the instructions
3. Launch from Start Menu or desktop shortcut

## How to Use

1. Select MKV files to convert
2. Choose output directory
3. Click "Convert"
4. Wait for conversion to complete

## System Requirements

- Windows 10 or later (64-bit)
- 4GB RAM
- 200MB+ free disk space
